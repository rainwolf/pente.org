BlueFrog is dead. Okopipi doesn't produce mucho despite some good ideas. I've been reporting to knujon which is nice but boring, and not nearly as effective as bluefrog was.

This latest script is more a framework to spread files using gmail in a tree-like network. It allows the root to spread digitally signed files through the network, to its children, and each child will automatically verify the digital signature, process the contents and spread it to their own children.

Children are stored in an encrypted file using the AES RijnDael (implemented by Dave Barton). Digital signatures are created using sha256 and ECDSA. Everything is written in Python and FreePascal and should compile/run on every platform where freepascal and python runs on (modulo some minor modifications to executable-calls from python).

You 'll need to install Python and libgmail to run this script. To compile the freepascal binaries you need FreePascal. Mail any suggestions to me, have fun
