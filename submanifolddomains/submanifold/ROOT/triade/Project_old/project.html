Project SCS

This project has been given the code name "SCS", which stands for Secure Communication Suite. Our aim is to provide the internet community with an application for linux to communicate over the internet in a secure way, fully compatible with PGP and provided with some extra functions, e.g. other public key algoritmes (ECDSA, ECElGamal, GOSTDSA, and so on) - secret key - or hash algorithms, extra functionality like a chat program using the DH key exchange protocol, a built in send mail - and receiver mail program. All this will be fully coded in FreePascal, and put together in a user-friendly user interface (not for the command line), similar to HD-COPY
  • If you are interested in this project, you can join it, we need more programmers. There are currently 5 participants.
  • We added some hyperlinks, that take you to the homepages of the different programs or libraries we use.
  • No source-code or binaries of SCS will be posted here before the first alpha/beta version is finished, we 'll post everything as soon as an alpha/beta version is available.
