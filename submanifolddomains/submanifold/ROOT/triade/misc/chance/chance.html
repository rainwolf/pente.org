It's like playing the lottery, but with a lower chance of winning. Whenever you feel like it, press enter, and the program generates random numbers, checks them for common factors with the RSA challenge numbers and that's it. It's written in FreePascal, I expect it to compile and run wherever FreePascal runs. The executable in this zip will only run on windows I 'm afraid. Should you be so lucky that one is found, then that factor will be written to the screen (quick, grab a pen and paper and write the number down, just in case...) and to a text file. The prize money ranges from $30000 to $200000. Aint that sweet?
