Export all object IDs
    IDs such as forum IDs, thread IDs and user IDs can be optionally exported. You may wish to do this if you want to preserve links your users may have to your community. For example, a user might bookmark:
    http://yoursite/jive/thread.jsp?forum=5&thread=293
    If you choose to not export IDs the above URL will not link to the same document after exporting then importing your data.
Export all users
    Optionally export users - this is separate from exporting their permissions. If you have a custom user implementation, it might be useful to not export your user base.
Export all groups
    Optionally export groups. You may wish to disable group exporting if you have a custom group implementation.
Export all permissions
    Optionally export permissions. If you choose not to export permissions, you'll need to re-create them after doing an import of the data.
Export Forums
    Optionally export forums. You may wish to skip exporting your forum data and just export users, for example.
Custom Filename
    You can specify a custom file name to export your data to. Note, only letters, numbers, "-", "_" and "." are allowed in the name of the file.
