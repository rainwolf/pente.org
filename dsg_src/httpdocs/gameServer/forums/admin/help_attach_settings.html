Max number of attachments per message
    The maximum number of attachments allowed per message.
Max attachment size
    The maximum size of each attachment, in kilobytes (K). A megabyte is 1024 kilobytes, or use the conversion table below.
    Size (K) Size (MB)
    256 0.25
    512 0.50
    768 0.75
    1028 1.00
    2056 2.00
    5140 5.00
Enable image previews
    Turns image previews on or off. If this feature is enabled, Jive Forums will automatically generate a small thumbnail image of the image attachment. If the feature is disabled, an image icon will be used instead.
Clear Image Cache
    Deletes all the generated thumbnail images.
Preserve image preview aspect ratio
    Preserves the aspect ratio (width:height) of a generated thumbnail image. If this feature is disabled, a square thumbnail will be produced. Note: this option is only available when image previews are enabled.
Max image preview dimension
    The maximum size of the longest side of the image preview. Note: this option is only available when image previews are enabled.
Content Types
    Every file has a content type, which describes the type of data a file contains. For example, a Windows executable file (.exe) has a content type of "application/octet-stream," and a GIF image has a content type of "image/gif." The content type settings allow you to control what kind of attachments users are allowed to post. For security reasons, you may wish to disallow certain types of attachments, such as EXE files.
Allow all content types
    When enabled, all content types are allowed except those that are put into a special "disallowed" list. This mode is useful when you want to allow a broad range of attachment types, but may need to protect against a certain few types of attachments being posted. To add content types to the "disallowed" list, select them from the list of common content types (using the "<<" button to add them), or use the "Add Custom Content Type" form.
Disallow all content types
    When enabled, no content types are allowed except those that are put into a special "allowed" list. This mode is useful when you want restrict the types of attachments users are allowed to post to a select few options. To add content types to the "allowed" list, select them from the list of common content types (using the "<<" button to add them), or use the "Add Custom Content Type" form.
